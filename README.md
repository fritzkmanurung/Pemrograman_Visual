# Pemrograman Visual (Pemvis) Course Work

Repository ini berisi kumpulan tugas, kuis, dan ujian untuk mata kuliah **Pemrograman Visual (2122)**. Kode ditulis menggunakan bahasa pemrograman Java.

## Struktur Repository

Proyek ini dikelompokkan ke dalam kategori berikut untuk memudahkan navigasi:

*   **[Exams/](./Exams/)**: Berisi kode untuk ujian praktikum (X01, X02).
*   **[Fundamentals/](./Fundamentals/)**: Berisi tugas dasar-dasar Git dan penggunaan GitHub.
*   **[Quizzes/](./Quizzes/)**: Berisi solusi untuk kuis-kuis yang diberikan selama perkuliahan.
*   **[Tasks/](./Tasks/)**: Berisi tugas mingguan (T00 - T06) yang mencakup berbagai topik pemrograman visual.

## Teknologi yang Digunakan
- Java SE
- Git/GitHub untuk Version Control

## Aktivitas Komit
Repository ini dikelola dengan komit bertahap (incremental commits) untuk mencerminkan progres belajar pada setiap modul tugas.
